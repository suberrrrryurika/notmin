# Notmin

一个轻量级的桌面化任务与笔记终端（前端）。本仓库包含前端界面与简单的任务编辑器，支持将任务记录并可设置提醒时间与重复频率。

## 功能亮点

- 暖白主题，使用统一的 CSS 变量，便于后续主题切换与维护。
- 侧边栏编辑器以覆盖层（overlay）弹出，不会导致主内容布局抖动。
- 简单的笔记/任务列表，支持 datetime 和频率（一次/每日/每周）调度。
- 在自定义标题栏与拖拽区域内优化了表单交互（确保 input/select 可用）。

## 快速开始（仅前端预览）

1. 安装依赖：
   - cd frontend
   - npm install
2. 快速在浏览器中预览（Windows）：
   - start frontend/index.html
3. 或在开发模式下运行（若项目配置了 dev 脚本）：
   - npm run dev

## 在 Tauri 中运行（桌面）

1. 确保已安装 Rust 与 Tauri 的依赖。
2. 在项目根目录运行：
   - cargo tauri dev
     或使用项目中配置的 npm 脚本（若有）。

## 本次 UI/样式清理要点

- 主题色与核心变量统一移动至 frontend/src/style.css 的 :root。
- 精简并合并重复的 CSS 规则以便维护。
- 将侧边栏改为覆盖式（overlay），并将遮罩限制为仅覆盖主内容，避免遮罩阻挡侧边栏内部弹出控件。
- 通过设置 -webkit-app-region: no-drag 并确保 pointer-events 可用，修复了标题栏拖拽区域导致的表单不可交互问题。
- 调整 z-index，使原生下拉（select）等弹出层能正确显示在遮罩之上。
- 打开侧边栏时会自动聚焦主输入框，提高使用体验。

## 使用检查清单（快速验证）

- 点击左上角菜单图标（≡），侧边栏应以覆盖方式弹出并自动聚焦到输入框。
- 在侧边栏中打开“重置频率”下拉，确认下拉能正常展开并选择项。
- 点击遮罩或再次点击菜单，侧边栏应关闭并移除遮罩。
- 在最大化/全屏状态下，标题栏的窗口控制按钮仍应保持可点击。
